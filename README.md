# 🚀 Sandy Admin Dashboard UI

A clean and modern **Admin Dashboard UI** built using **HTML, CSS, and JavaScript**.  
This project demonstrates responsive design, dark mode functionality, and interactive UI elements.

## ✨ Features
- 📊 Dashboard analytics cards (Sales, Expenses, Income)
- 🌙 Dark / Light mode toggle
- 📱 Fully responsive design (mobile + desktop)
- 📂 Sidebar navigation menu
- 📈 Recent orders & updates section
- 🎨 Clean UI with modern CSS variables

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (DOM Manipulation)
